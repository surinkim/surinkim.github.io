## GitHub Trending


- [mvanhorn/last30days-skill](https://github.com/mvanhorn/last30days-skill)<br>Reddit, X, YouTube, HN, Polymarket 등 웹 전반을 병렬 검색해 실제 사람들의 upvote·like·베팅 데이터로 점수를 매기는 AI agent 검색 skill입니다. 에디터가 아닌 대중의 관심과 돈으로 순위가 정해지며, 어떤 주제든 근거 있는 요약을 생성합니다. Claude Code, Codex, Cursor 등 50여 개 Agent Skills 호스트에서 zero-config로 동작합니다.



- [apple/container](https://github.com/apple/container)<br>Apple silicon에 최적화된 Swift 기반 도구로, Mac에서 Linux 컨테이너를 경량 가상머신으로 생성·실행합니다. OCI 호환 이미지를 사용해 표준 레지스트리에서 이미지를 pull/push할 수 있습니다. macOS 26의 가상화·네트워킹 기능을 활용하며 apple/containerization Swift 패키지를 기반으로 합니다.



- [phuryn/pm-skills](https://github.com/phuryn/pm-skills)<br>제품 관리(PM)를 위한 68개 skill과 42개 chained workflow를 9개 plugin으로 묶은 마켓플레이스입니다. discovery, 전략, 실행, 출시, 성장까지 검증된 PM 프레임워크(Teresa Torres, Marty Cagan 등)를 단계별로 안내합니다. Claude Code와 Cowork에 맞춰 설계되었고 다른 AI assistant와도 호환됩니다.




- [NVIDIA/SkillSpector](https://github.com/NVIDIA/SkillSpector)<br>AI agent skill을 설치하기 전에 취약점, 악성 패턴, 보안 위험을 탐지하는 보안 스캐너입니다. prompt injection, 데이터 유출, 권한 상승 등 16개 카테고리 64개 취약점 패턴을 정적 분석과 선택적 LLM 의미 분석으로 검사합니다. Git repo, URL, zip, 디렉터리 등 다양한 입력을 지원하고 0~100점 risk score를 제공합니다.



- [addyosmani/agent-skills](https://github.com/addyosmani/agent-skills)<br>AI 코딩 agent를 위한 production급 엔지니어링 skill 모음입니다. spec → plan → build → test → review → ship으로 이어지는 개발 라이프사이클을 7개 slash command로 매핑합니다. 시니어 엔지니어의 워크플로와 품질 게이트를 agent가 일관되게 따르도록 패키징했습니다.



- [Panniantong/Agent-Reach](https://github.com/Panniantong/Agent-Reach)<br>AI agent에 인터넷 접근 능력을 한 번에 부여하는 CLI 도구입니다. Twitter, Reddit, YouTube, GitHub, Bilibili, 샤오홍슈(小红书) 등을 별도 API 비용 없이 읽고 검색할 수 있습니다. 가장 안정적인 접근 방식을 자동으로 선택·설치·점검해 주어 접근 방식이 바뀌어도 신경 쓸 필요가 없습니다.



- [openai/plugins](https://github.com/openai/plugins)<br>Codex plugin 예제를 큐레이션한 저장소입니다. 각 plugin은 .codex-plugin/plugin.json 매니페스트와 함께 skills, MCP, commands, hooks 등 부가 요소를 포함합니다. figma, notion, build-ios-apps, expo, netlify 등 풍부한 실전 예제를 제공합니다.
